# Browser Support

Extraction UI requires **modern browsers with support for OKLCH colors and the color-mix() function**.

## Supported Browsers

| Browser        | Version | Year |
| -------------- | ------- | ---- |
| Chrome         | 111+    | 2023 |
| Firefox        | 113+    | 2023 |
| Safari         | 16.4+   | 2023 |
| Edge           | 111+    | 2023 |
| iOS Safari     | 16.4+   | 2023 |
| Chrome Android | 111+    | 2023 |

## Required Features

- **ES2020 JavaScript**
- **CSS Grid & Flexbox**
- **CSS Variables**
- **CSS Cascade Layers**
- **OKLCH color format** (for color palette generation)
- **color-mix() function** (for dynamic color mixing)

## Framework Support

- **React**: 19.2.0+
- **React DOM**: 19.2.0+
- **Next.js**: 15.0+ (recommended)

## Notes

- **Server-Side Rendering:** Fully compatible. All styles are static CSS with no JavaScript calculations.
