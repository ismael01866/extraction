# Browser Support

## Supported Browsers

| Browser        | Version | Year |
| -------------- | ------- | ---- |
| Chrome         | 147+    | 2026 |
| Firefox        | 146+    | 2025 |
| Safari         | 26+     | 2025 |
| Edge           | 147+    | 2026 |
| iOS Safari     | 26+     | 2025 |
| Chrome Android | 147+    | 2026 |

## Required Features

- **ES2020 JavaScript**
- **CSS Grid & Flexbox**
- **CSS Variables**
- **CSS Cascade Layers**
- **color-mix() function** (for dynamic color mixing)
- **color-contrast() function** (for dynamic text color contrast)

## Framework Support

- **React**: 19.2.0+
- **React DOM**: 19.2.0+
- **Next.js**: 15.0+ (recommended)

## Notes

- **Server-Side Rendering:** Fully compatible. All styles are static CSS with no JavaScript calculations.
