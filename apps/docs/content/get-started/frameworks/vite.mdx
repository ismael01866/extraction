# Vite

import { Steps } from 'nextra/components';

This guide is specific to Vite with React. For official framework documentation, visit the [Vite Documentation](https://vite.dev/guide/).

## Prerequisites

Before you begin, make sure your app is using:

- **Vite** 6 or higher

- **React** 19.2.0 or higher
- **React DOM** 19.2.0 or higher

## Installation

<Steps>

### Create a Vite app

```bash
pnpm create vite@latest --template react-ts
```

### Install the package

```bash
pnpm install extraction-ui
```

### Install peer dependencies

Extraction UI requires Radix UI as a peer dependency:

```bash
pnpm add radix-ui
```

And Tailwind CSS for styling:

```bash
pnpm add -D tailwindcss @tailwindcss/vite
```

Add the Tailwind plugin to your Vite config:

```ts
/** vite.config.ts */
import { defineConfig } from 'vite';

import tailwindcss from '@tailwindcss/vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
  plugins: [react(), tailwindcss()],
});
```

For more details on setting up Tailwind CSS, refer to the [Tailwind CSS Installation Guide](https://tailwindcss.com/docs/installation).

### Import styles in `src/index.css`

Create a global stylesheet and add the Extraction UI styles:

```css
/** src/index.css */
@import 'tailwindcss';
@import 'extraction-ui/styles';
```

Make sure the CSS file is imported in your entry point:

```tsx
/** src/main.tsx */
import { StrictMode } from 'react';

import { createRoot } from 'react-dom/client';

import './index.css';

import App from './App.tsx';

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <App />
  </StrictMode>,
);
```

### Start using components

Import and use components in your project:

```tsx
/** src/app.tsx */
import { Button, Card } from 'extraction-ui';

export default function App() {
  return (
    <Card>
      <Card.Header>
        <Card.Title>Welcome to Extraction UI</Card.Title>
      </Card.Header>
      <Card.Content>
        <Button className='mbs-4'>Submit</Button>
      </Card.Content>
    </Card>
  );
}
```

</Steps>
