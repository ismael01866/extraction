## Placement

Use the `side` prop on the `HoverCard.Content` to set the placement of the component. Values can include `top`, `right`, `bottom`, `left`.

{['top', 'right', 'bottom', 'left'].map((placement, index) => (

  <div key={placement}>
    { index > 0 ? <hr className='x:my-8 nextra-border' /> : <br /> }
    <Usage
      key={placement}
      components={{
        usage: React.cloneElement(PlacementComponent({ placement })),
        wrapper: ({ children }) => (
          <HoverCard>
            <HoverCard.Trigger asChild>
              <Button>Hover me ({placement})</Button>
            </HoverCard.Trigger>
            <HoverCard.Portal>{children}</HoverCard.Portal>
          </HoverCard>
        ),
      }}
    />
  </div>
))}
