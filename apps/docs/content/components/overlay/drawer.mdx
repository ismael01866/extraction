## Placement

Use the `placement` prop on the `Drawer.Content` to set the placement of the component. Values can include `start`, `end`, `top`, `bottom`.

{['start', 'end', 'top', 'bottom'].map((placement, index) => (

  <div key={placement}>
    { index > 0 ? <hr className='x:my-8 nextra-border' /> : <br /> }
    <Usage
      key={placement}
      components={{
        usage: React.cloneElement(PlacementComponent({ placement })),
        wrapper: ({ children }) => (
          <Drawer>
            <Drawer.Trigger asChild>
              <Button>Trigger ({placement})</Button>
            </Drawer.Trigger>
            <Drawer.Portal>
              <Drawer.Overlay />
              {children}
            </Drawer.Portal>
          </Drawer>
        ),
      }}
    />
  </div>
))}
