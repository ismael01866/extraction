## Placement

Use the `side` prop on the `Tooltip.Content` to set the placement of the component. Values can include `top`, `right`, `bottom`, `left`.

{['top', 'right', 'bottom', 'left'].map((placement, index) => (

  <div key={placement}>
    { index > 0 ? <hr className='x:my-8 nextra-border' /> : <br /> }
    <Usage
      key={placement}
      components={{
        usage: React.cloneElement(PlacementComponent({ placement })),
        wrapper: ({ children }) => (
          <Tooltip.Provider>
            <Tooltip>
              <Tooltip.Trigger>
                <Button>Hover me ({placement})</Button>
              </Tooltip.Trigger>
              <Tooltip.Portal>{children}</Tooltip.Portal>
            </Tooltip>
          </Tooltip.Provider>
        ),
      }}
    />
  </div>
))}
