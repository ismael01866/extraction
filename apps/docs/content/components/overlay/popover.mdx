## Placement

Use the `side` prop on the `Popover.Content` to set the placement of the component. Values can include `top`, `right`, `bottom`, `left`.

{['top', 'right', 'bottom', 'left'].map((placement, index) => (

  <div key={placement}>
    { index > 0 ? <hr className='x:my-8 nextra-border' /> : <br /> }
    <Usage
      key={placement}
      components={{
        usage: React.cloneElement(PlacementComponent({ placement })),
        wrapper: ({ children }) => (
          <Popover>
            <Popover.Trigger asChild>
              <Button>Click me ({placement})</Button>
            </Popover.Trigger>
            <Popover.Portal>{children}</Popover.Portal>
          </Popover>
        ),
      }}
    />
  </div>
))}
