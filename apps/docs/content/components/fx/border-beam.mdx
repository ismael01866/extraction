## Customizing

```sass filename="globals.css"
:root {
  --ex-border-beam-color: transparent, var(--color-primary), transparent;

  --ex-border-beam-width: 1px;
  --ex-border-beam-length: 50%;

  --ex-border-beam-radius: var(--radius-sm);

  --ex-border-beam-delay: 0s;
  --ex-border-beam-ease: linear;
  --ex-border-beam-duration: 20s;

  --ex-border-beam-fill: forwards;
  --ex-border-beam-direction: normal;

  --ex-border-beam-count: infinite;
  --ex-border-beam-play: running;

  --ex-border-beam-animation: border-beam-pulse;
}
```
