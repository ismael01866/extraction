## Basic Animation

Use the `activeClassName` and `inactiveClassName` props to apply CSS classes when the element is in or out of view.

<Usage
  components={{
    usage: ComponentBasic,
    wrapper: ({ children }) => (
      <ScrollArea className='h-20 max-h-40 w-full' type='always'>
        <ScrollArea.Viewport>
          <ScrollArea.Content>
            <div>
              <div className='mb-20'>Scroll to reveal animation</div>
              {children}
            </div>
          </ScrollArea.Content>
        </ScrollArea.Viewport>
        <ScrollArea.Scrollbar>
          <ScrollArea.Thumb />
        </ScrollArea.Scrollbar>
      </ScrollArea>
    ),
  }}
/>

## Render Props

Use the `isInView` render prop to access the visibility state directly. <br /> If you are using RSC remember to add the `use client` directive when accesing render props.

<Usage
  components={{
    usage: <InViewRenderPropExample />,
    wrapper: Wrapper,
  }}
  serializedCode={`<ScrollArea className="h-20 max-h-40 w-full" type="always">
    <ScrollArea.Viewport>
      <ScrollArea.Content>
        <div>
          <div className="mb-20">Scroll to reveal animation</div>
          <InView>
            {({ isInView }) => (
              <Counter key={isInView ? 'in' : 'out'} value={1000} />
            )}
          </InView>
        </div>
      </ScrollArea.Content>
    </ScrollArea.Viewport>
    <ScrollArea.Scrollbar>
      <ScrollArea.Thumb />
    </ScrollArea.Scrollbar>
  </ScrollArea>`}
/>
