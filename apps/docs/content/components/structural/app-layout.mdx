## Customizing

```sass filename="globals.css"
:root {
  --ex-app-layout-px: calc(var(--spacing) * 5);
  --ex-app-layout-py: calc(var(--spacing) * 5);

  --ex-app-layout-header-min-height: calc(var(--spacing) * 16);
  --ex-app-layout-footer-min-height: calc(var(--spacing) * 16);

  --ex-app-layout-aside-width: calc(var(--spacing) * 64);

  --ex-app-layout-sidenav-gap-y: calc(var(--spacing) * 4);
  --ex-app-layout-sidenav-width: calc(var(--spacing) * 64);
}
```
