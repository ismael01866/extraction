# Blur

Blur tokens control the amount of gaussian blur applied.
For more information on the default available blur utilities, visit [Tailwind](https://tailwindcss.com/docs/blur)

```css
@theme {
  --blur-xs: 1px;
  --blur-sm: 2px;
  --blur-md: 4px;
  --blur-lg: 8px;
  --blur-xl: 12px;
  --blur-2xl: 16px;
  --blur-3xl: 24px;
}
```
