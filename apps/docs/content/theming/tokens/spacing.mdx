# Size & Space

The base spacing unit is `0.25rem` (4px), assuming a root font size of 16px. Every Tailwind spacing utility — `p-`, `m-`, `gap-`, `w-`, `h-`, etc. — multiplies this value by its scale step.

```css
@theme {
  --spacing: 0.25rem;
}
```
