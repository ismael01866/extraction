# Line Height

Line height tokens control the vertical spacing between lines of text.
For more information on the default available line-height utilities, visit [Tailwind](https://tailwindcss.com/docs/line-height)

```css
@theme {
  --leading-tight: 1.25;
  --leading-snug: 1.375;
  --leading-normal: 1.5;
  --leading-relaxed: 1.625;
  --leading-loose: 2;
}
```
