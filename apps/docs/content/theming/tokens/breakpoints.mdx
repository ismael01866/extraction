# Breakpoints

Breakpoint tokens define the min-width thresholds used by responsive utilities. For more information on the default available breakpoints and utilities, visit [Tailwind](https://tailwindcss.com/docs/breakpoints)

```css
@theme {
  --breakpoint-sm: 40rem;
  --breakpoint-md: 48rem;
  --breakpoint-lg: 64rem;
  --breakpoint-xl: 80rem;
  --breakpoint-2xl: 96rem;
}
```
