# Radius

Border radius tokens control the roundness of component corners.
For more information on the default available border-radius utilities, visit [Tailwind](https://tailwindcss.com/docs/border-radius)

```css
@theme {
  --radius-xs: 0.125rem;
  --radius-sm: 0.25rem;
  --radius-md: 0.375rem;
  --radius-lg: 0.5rem;
  --radius-xl: 0.75rem;
  --radius-2xl: 1rem;
  --radius-3xl: 1.5rem;
  --radius-4xl: 2rem;
}
```
