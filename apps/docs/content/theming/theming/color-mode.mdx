# Color Mode

Color mode allows components to adapt their appearance between light and dark themes. <br/> Dark mode is enabled by adding either the `.dark` class or the `data-theme="dark"` attribute to the root `<html>` element.

Add the theme to your root HTML element:

```html
<html class="dark">
  ...
</html>
```

or:

```html
<html data-theme="dark">
  ...
</html>
```
