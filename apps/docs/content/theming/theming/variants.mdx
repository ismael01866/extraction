# Variants

Variants apply Extraction UI palette to components, ensuring consistent color usage and visual hierarchy across the interface.

Palettes define the foundational color scheme used by each variant (solid, outline, ghost, ...). They include specific values for background, border, outline, and text colors that work together to create consistent visual styles.

## Overview

The color palette is built on a semantic foundation where each color family serves a specific purpose:

- **Primary, Accent** — Brand colors for key actions and highlights
- **Neutral** — Backgrounds, borders, and text
- **Success, Warning, Error, Info** — Feedback and status messaging

## Colors

Extraction UI provides semantic color options that adapt to each component's variant styles. For creating your own palettes, see the [Creating Palettes](#creating-palettes) section below.
